# 给客户的安装和使用指南

## 📥 安装步骤

### 第一步：获取工具

1. 从提供的位置下载 `OpenClawGateway.exe` 文件
2. 将文件保存到任意位置（建议：`C:\Program Files\OpenClawGateway\` 或桌面）

### 第二步：首次运行

1. 双击 `OpenClawGateway.exe`
2. 如果出现 Windows 安全提示，点击"更多信息" → "仍要运行"
3. 程序会在系统托盘（右下角时钟旁）显示一个圆形图标

## 🎯 使用方法

### 查看图标状态

托盘图标颜色代表 OpenClaw Gateway 的状态：
- 🟢 **绿色**：Gateway 正在运行
- 🔴 **红色**：Gateway 已停止
- 🟡 **黄色**：正在执行操作（启动/停止/重启）

### 打开控制面板

**左键点击**托盘图标，会弹出控制面板，可以：
- 查看当前状态
- 启动/停止/重启 Gateway
- 打开网页控制台
- 查看运行日志

### 托盘菜单

**右键点击**托盘图标，可以：
- 启动/停止/重启 Gateway
- 打开网页控制台
- 设置开机自启动
- 退出程序

## ⚙️ 常用设置

### 设置开机自动启动

**方法一：通过控制面板**
1. 左键点击托盘图标打开面板
2. 勾选底部的"Start with Windows"复选框

**方法二：通过托盘菜单**
1. 右键点击托盘图标
2. 点击"Auto-start on Boot"（勾选表示已启用）

### 打开 Web 控制台

1. 点击面板中的"🌐 Open Web UI"按钮
2. 或右键托盘图标 → "Open Web UI"
3. 浏览器会自动打开 http://127.0.0.1:18789

## 🔧 常见问题

### ❓ 点击启动按钮没反应？

**解决方法：**
1. 右键托盘图标 → "View Debug Log"查看日志
2. 确认 OpenClaw 已正确安装（在 PowerShell 中输入 `openclaw` 测试）
3. 以管理员权限运行程序（右键 exe → "以管理员身份运行"）

### ❓ 图标一直是红色，但 Gateway 应该在运行？

**解决方法：**
1. 打开浏览器访问 http://127.0.0.1:18789
2. 如果能访问，说明 Gateway 正常，可能是工具检测延迟
3. 等待几秒，图标会自动更新为绿色
4. 或点击"重启 Gateway"按钮

### ❓ 如何卸载？

1. 右键托盘图标 → "Auto-start on Boot"取消勾选（关闭自启动）
2. 右键托盘图标 → "Exit"退出程序
3. 删除 `OpenClawGateway.exe` 文件

### ❓ 双击程序后没有窗口出现？

这是正常的！程序启动后会最小化到系统托盘（右下角时钟旁边），找到圆形图标即可。

## 📞 获取帮助

如果遇到问题：
1. 右键托盘图标 → "View Debug Log"查看详细日志
2. 将日志内容截图发送给技术支持

## 💡 使用技巧

1. **快速启动**：建议勾选"开机自启动"，这样电脑启动后就能使用
2. **状态检查**：鼠标悬停在托盘图标上可以看到当前状态
3. **快捷访问**：将程序固定到任务栏方便快速打开
4. **日志排查**：遇到问题时，第一时间查看 Debug Log

## ⚠️ 注意事项

1. **不要重复点击**：点击启动/停止/重启按钮后，请等待操作完成（图标变色）
2. **管理员权限**：某些操作可能需要管理员权限，建议以管理员身份运行
3. **端口占用**：确保 18789 端口没有被其他程序占用
4. **防火墙设置**：如果有安全软件拦截，请添加信任

## ✅ 快速开始

**最简单的使用流程：**

1. 双击运行 `OpenClawGateway.exe`
2. 找到系统托盘图标（右下角）
3. 左键点击图标打开面板
4. 点击"▶ Start"启动 Gateway
5. 等待图标变绿色
6. 点击"🌐 Open Web UI"开始使用

就这么简单！😊
