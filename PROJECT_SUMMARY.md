## 📊 技术选型说明

### 为什么选择 Python + Tkinter？

✅ **优点：**
- 开发快速，代码简洁
- Tkinter 是标准库，无需额外安装
- pystray 托盘功能完善
- 打包后可独立运行
- 跨平台（虽然本项目主要面向 Windows）

⚠️ **缺点：**
- 打包后体积较大（15-25MB，Nuitka 可优化到 8-15MB）
- 界面美观度不如 Electron
- 启动速度不如 C#/Go 原生应用

### 打包工具选择

| 工具 | 优点 | 缺点 | 推荐场景 |
|------|------|------|----------|
| PyInstaller | 快速、兼容性好 | 体积大、启动慢 | 快速开发、测试 |
| Nuitka | 体积小、速度快 | 构建慢、学习曲线 | 生产部署、追求性能 |

**建议：开发阶段用 PyInstaller，正式发布用 Nuitka**

## 🔧 已知限制和注意事项

### 限制

1. **仅支持 Windows** - 使用了 Windows 特定的功能（注册表、通知）
2. **需要 OpenClaw 预装** - 程序依赖 `openclaw` 命令行工具
3. **端口固定** - 默认检测 18789 端口，未提供配置选项
4. **单实例检测** - 未实现，可能运行多个实例

### 注意事项

1. **管理员权限** - 某些操作（如写入注册表）可能需要管理员权限
2. **防火墙** - 首次运行可能被防火墙拦截
3. **杀毒软件** - 打包的 exe 可能被误报，需添加白名单
4. **端口占用** - 18789 端口被占用时会检测失败

## 📞 技术支持建议

### 常见问题速查

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 图标不显示 | 程序未启动 | 检查任务管理器 |
| 启动失败 | openclaw 未安装 | 运行官方安装命令 |
| 自启动不生效 | 权限不足 | 以管理员运行 |
| 通知不弹出 | winotify 未安装 | `pip install winotify` |

### 收集诊断信息

1. 右键托盘图标 → "View Debug Log"
2. 点击"Refresh"
3. 复制所有日志
4. 检查 Windows 事件查看器
